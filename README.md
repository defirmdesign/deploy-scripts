# Scripts

## git-clone

This script sets up a dedicated SSH key and handles SSH config setting for adding the environment to the deploy keys of a repo.

```bash
# Download the git-clone.sh script
wget https://raw.githubusercontent.com/defirmdesign/deploy-scripts/main/scripts/git-clone.sh

# Make it executable
chmod +x git-clone.sh

# Run the script
./git-clone.sh
```
